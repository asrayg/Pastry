# Pastry 🥐

Windows-style clipboard history (Win+V) for macOS. Native Swift, zero dependencies.

Press **⌘⇧V** anywhere to see everything you've copied and paste any of it back.

## Usage

- Copy things as usual (⌘C). Text and images are recorded automatically.
- Press **⌘⇧V** to open the history panel near your cursor.
- **↑/↓** to select, **↩** to paste into the app you were in, or just click an item.
- **P** pins an item (pinned items survive "Clear All" and never age out), **⌫** deletes, **esc** closes.
- The panel also closes when it loses focus, like the Win+V flyout.
- Menu bar icon → Clear History, Start at Login, About, Quit.

Matches Windows behavior: keeps the 25 most recent items (pinned items don't
count against the limit), skips items over 4 MB, ignores content marked
concealed/transient by password managers, and re-copying an existing item
moves it back to the top.

## Permissions

On first launch macOS asks for **Accessibility** access
(System Settings → Privacy & Security → Accessibility). This is required to
synthesize the ⌘V keystroke that pastes the chosen item. Without it Pastry
still records history and copies your selection — you'd just press ⌘V yourself.

## Build

```sh
./build.sh          # produces build/Pastry.app (and regenerates the icon if needed)
open build/Pastry.app
```

To install permanently, copy `build/Pastry.app` to `/Applications` and
enable "Start at Login" from the menu bar icon.

## Notes

- Hotkey is defined in `Sources/Pastry/HotKey.swift` (`kVK_ANSI_V` + `cmdKey|shiftKey`).
- History is persisted to `~/Library/Application Support/Pastry/history.json`.
- The app icon is generated by `scripts/makeicon.swift` into `Resources/AppIcon.icns`.
- ⌘⇧V means "paste without formatting" in some apps (Docs, Slack); change the
  hotkey there if that bothers you.
